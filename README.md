Our Magento Payment Gateway by Novalnet enables secure integration of payments and payment services for Magento 1 shops all under one roof. One platform, simple integration and one contract partner from checkout to collection.
## Payment Integration requirement 
Novalnet <a href="https://www.novalnet.de/"> merchant account </a> is required for process payments through the Magento 1 Payment Gateway. The module is available for the 1.4.2.x-1.9.4.3 version in the following languages: EN & DE

## Installation
Follow these steps to install the Magento 1 Payment Integration by Novalnet:
1. Get the Magento 1 Payment Module & detailed documentation by <a href="https://www.novalnet.de/kontakt/sales"> contacting us </a>
2. Install the Module to your Magento 1 shop
3. Configure Product Activation Key in the shop admin panel
4. Activate & configure the preferred payment types in your shop admin panel
5. You’re ready to go!
